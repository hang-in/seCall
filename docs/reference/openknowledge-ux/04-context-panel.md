# 04. 우측 컨텍스트 패널

상단 아이콘 4탭: **Outline · Links · Graph · Timeline**. 현재 문서 기준 맥락을 항상 곁에.

## 4.1 Outline (아웃라인/TOC)
- 문서 헤딩 트리(예 `제목 > 섹션`), 클릭 시 해당 섹션 스크롤. 카운트 배지.
- **구현**: 저비용. 헤딩 파싱 → 클릭 스크롤.

## 4.2 Links (링크) — 백링크 ★P0
- **OUTGOING(n)**: 이 문서가 거는 링크 → 대상 문서명 + **컨텍스트 스니펫**(원문 라인).
- **BACKLINKS(n)**: 이 문서를 거는 문서 → 소스 문서명 + 스니펫.
- **왜**: 지식 연결의 핵심. 세션→위키에서 "이 결정이 어디서 참조됐나"를 즉시.
- **구현**: 문서별 링크 인덱스(outgoing/backlink) + 각 링크에 원문 라인 스니펫.

## 4.3 Graph (그래프)
- **로컬 force-directed 그래프**: 현재 문서 = 파랑 하이라이트 노드 + 연결 노드. 상단 `N nodes · N links`.
- **전역 토글(🌐)** = 전체 볼트 그래프, **전체화면(⤢)**.
- **구현**: seCall은 이미 Knowledge Graph 있음 → **문서 우측 로컬 그래프**로 재사용(전역만 있으면 로컬 컨텍스트 뷰 추가).

## 4.4 Timeline (버전 히스토리)
- 문서별 버전 타임라인(체크포인트/restore). 새 문서는 "No history yet".
- CRDT 기반(→ [07](07-collab-versioning-sharing.md)).
- **구현**: seCall이 git 동기화라 히스토리 있음 → 문서별 타임라인 UI로 노출.
